🚀 Quick Start

Step 1: Upload Files to Public HTML

Upload all files to your public_html folder on Hostinger:

index.html
style.css
script.js
config.php
database.sql
.htaccess
debug.php (optional - for troubleshooting)

Step 2: Create Database

Using Hostinger's phpMyAdmin:

  1. Go to cPanel → phpMyAdmin
  2. Create a new database (if not already created)
  3. Import the database.sql file into your database

Step 3: Verify Configuration

Visit your domain to check if everything is working:

https://note.websweos.com/debug.php

Note: Remove debug.php from production for security.

🛠️ Troubleshooting

Problem: "Database connection failed"

Solution:

  • Visit debug.php to check database connection
  • Verify credentials in config.php
  • Ensure database user has proper permissions
  • Check if database exists in phpMyAdmin

Problem: "Login/Register not working"

Solution:

  • Check browser console (F12) for JavaScript errors
  • View Network tab to see API responses
  • Ensure api/auth.php is accessible
  • Check error.log file in root directory

Problem: "CORS errors or 404 errors"

Solution:

  • Verify .htaccess file is in root directory
  • Check API file paths are correct
  • Ensure mod_rewrite is enabled on your server

Problem: "Blank page or white screen"

Solution:

  • Check PHP error log: error.log in root
  • Visit debug.php for detailed diagnostics
  • Check server console.txt in Hostinger
  • Verify all PHP files have correct syntax

🔒 Security Considerations

  • Remove debug.php from production
  • Update JWT_SECRET in config.php to a unique value
  • Use HTTPS for all connections
  • Implement rate limiting to prevent brute force attacks
  • Use environment variables for sensitive data instead of hardcoding
  • Validate all user inputs on both client and server side
  • Enable CSRF protection for form submissions
